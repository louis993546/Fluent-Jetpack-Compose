# Fluent Theme for Android (powered by Jetpack Compose)

If you want to build an app complient with Fluent Design, using Jetpack Compose, this would be your
solution.

## Example

```kt
@Composable
fun YourApp() {
    FluentTheme {
        Column {
            Text(text = "Hello, Fluent World!")
        }
    }
}
```
